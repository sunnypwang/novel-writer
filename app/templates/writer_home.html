The Project

Jane + AI = JAIne

JAIne is a custom trained GPT-2 model that learned how to generate a fanfiction for Pride and Prejudice, orignally written by Jane Austen.

Interactive

JAIne will continue a story based on the prompt you enter. Try different prompts for different stories!

Entertaining

JAIne readS the whole Pride and Prejudice book before learning how to write one. It knows how to amuse readers!

Extensions

Some extensions to this project include adapting it to modern novels to reimagine other stories in a Victorian Era English style.
